# Film_Database

This respository contains all files, scripts and documentation for a Film Database Application for the Applied Databases module project.

---

### Objective

- Work with a relational database (moviesDB) and query it using MySQL.
- Work with a non-relational database (movieScriptsDB) query it using MongoDB.
- Writing a Python application allowing the user to interact with both databases via CRUD operations. 

---

 ## Repository Contents
 
- ```Innovation``` - a folder detailing extra functionality added to the application outside of the project brief.
- ```Mongo-Queries``` - a folder containing mongoDB queries for exercise questions.
- ```MYSQL-Queries``` - a folder containing SQL queries for exercise questions.
- ```PythonApp``` - a folder containing the Python script for the user interface to interact with the databases.
- ```Project Brief``` - Python App & Innovation.pdf`` - the project brief.
- ```SQL & MongoDB Query Questions.pdf``` - the questions for ```Mongo-Queries``` and  ```MYSQL-Queries```.
- ```moviesDB.sql``` - the SQL database.
